Constructor to create a matrix and initialize it to the elements given by a vector.

The elements should be given in flattened 1-dimensional form, with the matrix element at row i and column j (counting from zero) given by element number (cols * i) + j of the vector (also counting from zero), where cols is the number of columns, that is, the number of element in each row. For example, for a 2x2 matrix A, the vector will be [A(0, 0), A(0, 1), A(1, 0), A(1, 1)].

Parameters
input_rowsThe number of rows.
input_colsThe number of columns.
input_elementsA vector containing the elements in flattened 1-dimensional form.
Exceptions
zero_sizeif the number of rows or columns is zero.
initializer_wrong_sizeif the size of the vector does not equal the total number of matrix elements.

The elements should be given in flattened 1-dimensional form, with the matrix element at row i and column j (counting from zero) given by element number (cols * i) + j of the initializer_list (also counting from zero), where cols is the number of columns, that is, the number of element in each row.
